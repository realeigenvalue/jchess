# jchess

Two player Java chess game using MVC design.

![Application Image](jchess.png)

## Building

Prerequisites
- Java JDK
- Gradle Build Tool

```bash
git clone https://github.com/realeigenvalue/jchess.git
cd jchess
gradle build
gradle run
```
